Comment Routes

POST /v1/api/comment/:postId

Description: Post a comment on a specific post.

URL: /v1/api/comment/{postId}

Method: POST

Headers:

authorization: <token>
x-client-id: <client_id>

Request Body:

{
    "room_id": "string (required)",
    "comment": "string (required)"
}

Responses:

200 OK:
{
    "message": "Comment posted successfully",
    "status": 200,
    "metadata": {
        "user_id": "string",
        "comment": "string",
        "post_id": "string",
        "room_id": "string",
        "created_at": "timestamp",
        "user_name": "string",
        "user_avatar": "string (url)"
    }
}

GET /v1/api/comment/:postId

Description: Retrieve a list of comments for a specific post.

URL: /v1/api/comment/{postId}

Method: GET

Headers:

x-client-id: <client_id>
authorization: <token>

Responses:

200 OK:
{
    "message": "Comment listed successfully",
    "status": 200,
    "metadata": [
        {
            "user_id": "string",
            "comment": "string",
            "post_id": "string",
            "room_id": "string",
            "created_at": "timestamp",
            "user_name": "string",
            "user_avatar": "string (url)"
        }
    ]
}
